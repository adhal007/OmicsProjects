Welcome to the documentation of OmixHub 

OmixHub is designed to be a platform to apply complex bioinformatics, machine learning or AI methods on large-scale omics data. 
It provides functionality for data retrieval from genomic data common's publicly available datasets, running bioinformatics or ML algorithms.
